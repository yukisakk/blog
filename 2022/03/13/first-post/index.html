自己紹介

29 歳ソフトウェアエンジニア、大学卒業してからプログラミングに携わって、かれこれ 3 年くらい。

そもそも、プログラミングを始めたきっかけは何ぞやっていう話なのだけれども、理由はシンプルで、就活すんのﾒﾝﾄﾞｸｾｯ ⇒ せや、自分で稼げばええやんっていう短絡的な思考から導き出された結果だったりする。

いったい、当時は絶賛金欠ﾏﾝだったので、初期投資が最小かつ、飯が食えるレベルで稼げる方法を知るために、chrome で『元手がかからない　仕事』って検索して、トップにプログラミングを見つけ……

プログラミング…..?

プログラミングのプの字も知らなかった僕は、とりあえず『プログラミングとは』って調べたのち、

めちゃくちゃ画面をいっぱい開いて、キーボードをｶﾀｶﾀ打っている人たちがおこなっていることだと知る。

って感じだったのだけれども、一度決めたからにはとりあえず取り組んでみるかーってことで、一番手が付けやすそうな、HTML + CSS ってやつをやることに決める。

本屋に立ち寄って、入門編と書かれた HTML 本をぱらぱらとめくってみるが、

「これほんまに入門やろか……😓 でもやるしかないかー」

ってことで、なけなしのお金で HTML+CSS の本を購入して、中古の MacBook Air も購入。

そして、僕の Web プログラミングはじまったのです。

遍歴

始めの頃は、

それでも、大学 1 年間を犠牲にして習得したキーボード入力スピード(MMO 仕込み)を駆使してひたすら写経。食べる暇も惜しんで写経。

血便まではいかなかったけど、栄養失調でぶっ倒れるくらいまで写経。

そういった経験があったからこそ、初めて Hello World!の文字がブラウザに表示されたとき、すっごく感動したのを今でも覚えている。

んで、その流れで JavaScript を経てぇ、

ずいぶん駆け足になったけど、プログラミングに関してはこんな感じ。

ここまで自分がたどってきたプログラミングの軌跡をざっくり述べてきたけど、実際、プログラミングが好きかと言われれば、それほど好きではない。それが、OSS 活動に対する姿勢として表れてしまっているのは直すべきなのかなぁと思っている反面、純粋に文章を書く方が好きだったりするという事実。

文章書く感じで、自分が独学してきたときに詰まったところとかを、備忘録的にまとめられれば良いかなと思ったりする。
